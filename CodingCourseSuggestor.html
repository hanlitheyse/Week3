I am here to share more about coding and courses

Coding is how we are able to interact with computers. In order for a computer to understand our ideas and solutions we need to translate it into a language that it understands. We need translators for computers to understand our human solutions and that is where we use programming languages like JavaScript, Java or Python.

Coding is used for websites, apps and multiple different other technologies and software that we interact with every day.
Frontend developers bring us the interface we interact with. Backend developers specialize in the parts that we cannot see - server-side software. Fullstack developers are able to create the interface and the server-side aspects.
